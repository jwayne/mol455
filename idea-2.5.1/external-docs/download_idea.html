Download IDEA (54 MB)

Instructions:
  1. Save the file in the directory where you would like to install IDEA.  We suggest creating a new directory for this purpose.
  2. From that directory, unpack the compressed archive using the command below:*

    tar xzf idea-2.5.1.tar.gz

  3. Be sure all required programs (see "Required Software" below) are in your path.
    4. If you are a Mac OS X user, be sure to install XCode before installing IDEA.
    You will need to run the installer from the bash shell.  If necessary, run bash to start it.
  4. Run the program install-idea to install IDEA.
  5. If installation is aborted for any reason, it is recommended that you delete all files except idea-2.5.1.tar.gz and repeat steps 2-4.
* Solaris users may need to substitute gtar xzf idea-2.5.1.tar.gz in order to run GNU tar.

System Requirements

www
Supported Operating Systems:
  • Linux
  • Solaris
  • Mac OS X 10.4 and higher with XCode, a free download from Apple
Hardware Requirements:
  • About 205MB of disk space
  • 1.28 GB RAM recommended
Required Software:
  • Java 1.5 or higher
  • Perl 5.6 or higher and the modules Log::Log4perl, Config::IniFiles and Bio::Tools::Phylo::PAML
  • PAML 4.4, 4.3, 4b, 4a, 3.15 or 3.14
  • PhyML 3.0 and/or PHYLIP (PhyML is not supported on Solaris.)
  • bash
  • Either SGE or Condor is required if support for grid computing is desired.  Otherwise, all processing occurs on the machine where IDEA is launched.  IDEA can execute processes in parallel on a multiprocessor machine.
If you need assistance installing any of these programs or modules or verifying that they are installed, please contact your systems administrator.
